Fine Tuning Llama 2

Description:

I enhanced a 7B general-purpose LlaMa 2 model for precise natural language to code conversion, streamlining pipelines for efficient processing of over 500K text inputs and code snippets to enable properly formatted prompts for training. I employed advanced techniques such as 4-bit quantization and parameter-efficient fine-tuning, optimizing model efficiency and reducing computational requirements. Utilizing Nvidia CUDA for hardware-accelerated model training significantly reduced fine-tuning time compared to CPU-based training.

Predictive Analytics in NBA

Description:

I devised an NBA shot prediction system using 5 million lines of historical game data, incorporating close to 20 metrics. Diabetes Patient Analysis

Description:

I performed extensive analysis on the Diabetes Dataset, which contains over 50 features and millions of lines of patient data. I compared Lloyd's K-means algorithm with various Gaussian Mixture Models, developing the Expectation-Maximization algorithm for GMM from scratch to ensure efficient data processing. Additionally, I tested the data on Poisson Mixture Models. The data was classified into appropriate clusters based on patients' medical data.

Speech Emotion Recognition

I established a deep learning model with TensorFlow and Keras, achieving 82% accuracy in categorizing human emotions from speech audio. I designed a custom data pipeline for acoustic feature extraction, reducing false positives and capturing emotional cues. The architecture, formulated using LSTM, CNN, and linear regressors, boosted accuracy by 15% compared to existing models. I presented these findings at the IEEE conference "2022 6th International Conference on Trends in Electronics and Informatics," highlighting the impact on emotion recognition technology.

Toxic Chat Detection

I developed a toxic speech detection model with NLTK, SpaCy, and linear regression, delivering over 95% accuracy. I established a vector space using TF-IDF scores and extracted features using techniques such as POS tagging, N-grams, entity recognition, and word embeddings, improving model accuracy by over 15%. I trained a recurrent neural network in combination with a logistic regressor to categorize words, sentences, and paragraphs based on toxicity levels, achieving an accuracy score of 95%.
